# Chainguard Actions

Hardened GitHub Actions built from source and continuously scanned to protect your CI/CD pipelines from supply chain attacks.

Every action includes SBOM and provenance attestations, and is designed to prevent common threats like tag hijacking, dependency confusion, `pull_request_target` abuse, and secret exfiltration.

Learn more at [chainguard.dev/actions](https://www.chainguard.dev/actions).

## Usage

```yaml
- uses: chainguard-actions/<action-name>@main
```
